# Input 输入框

表单中的输入框组件

基本用法

点击查看代码

通过v-model双向绑定数据

<template>
  <bu-input placeholder='请输入内容' v-model='value'></bu-input>
</template>

<script>
  export default{
    data(){
      return {
        value:''
      }
    }
  }
</script>

可清空

点击查看代码

通过clearable得到一个可清空的输入框

<template>
  <bu-input placeholder='请输入内容' v-model='value' clearable></bu-input>
</template>

密码框

点击查看代码

通过showPassword属性控制密码的可视与否

<template>
  <bu-input placeholder='请输入密码' v-model='value'  showPassword></bu-input>
</template>
