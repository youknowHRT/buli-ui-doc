# Dialog 对话框

在保留当前页面状态的情况下，告知用户并承载相关操作。

基本用法

Dialog弹出一个对话框，处理定制任务场景。
点击查看代码

需要设置visible属性，它接收Boolean，当为true时显示 Dialog。Dialog 分为三个部分：title、body和footer，title和footer分别需要具名为title以及footer的slot，title定义标题。

<template>
    <div>
      <bu-button @click="visible=true" class="showDialog">点击弹出对话框</bu-button>
    </div>
    <bu-dialog  :visible.sync="visible">
        <template v-slot:title>
          <span>
            精灵选择
          </span>
        </template>
        <p>已选择蒜头王八作为主宠物</p>
        <template v-slot:footer>
          <bu-button type='primary' @click="visible=false">确认</bu-button>
          <bu-button @click="visible=false">取消</bu-button>
        </template>
      </bu-dialog>
  </template>
