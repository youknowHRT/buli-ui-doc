# Pagination 分页器

当数据量过多时，使用分页分解数据。

基本用法

点击查看代码

totalPage属性设置页面数量，currentPage设置当前页面

<template>
  <p>页数较少的时候</p>
  <bu-pagination :totalPage="7" :currentPage.sync="current"/>
  <p>页数大于7的时候</p>
  <bu-pagination :totalPage="20" :currentPage.sync="current2"/>
</template>
<script>
  export default{
    data(){
      return {
        current:1,
        current2:1
      }
    }
  }
</script>

当只有一页时隐藏分页

点击查看代码

hideIfOnePage绑定boolean类型，为可选属性。默认是显示的，当总页数为1且属性为true时，组件display为none

<template>
  <bu-pagination :totalPage="1" :currentPage.sync="current" hideIfOnePage/>
</template>
