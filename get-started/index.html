    # 快速上手

    注意

    本UI组件库仅供学习交流。✨ 😃

    # 安装

    推荐使用npm / yarn来安装。

    npm install buli-ui
<!-- or  -->
yarn add buli-ui

    # 使用之前

    在使用buli-ui之前，假设您已经了解Vue、Vue组件、Vue单文件组件的基础知识。

    # 使用构建工具

    一般在webpack入口页面main.js中如下配置：



     
     
     







    import Vue from 'vue'
import App from './App.vue'
import BuliUi from 'buli-ui'  //引入组件库
import 'buli-ui/dist/buli-ui.css'  //引入样式文件
Vue.use(BuliUi)  //全局使用

Vue.config.productionTip = false

new Vue({
  render: h => h(App)
}).$mount('#app')
